# SkillShare - Community Skills Exchange Platform

A web-based platform for exchanging skills and knowledge without money.

## Tech Stack
- Frontend: Next.js 15, React 19, TailwindCSS
- Backend: Strapi v5
- Database: SQLite
- Auth: JWT
- Email: Nodemailer

## How to Run

### Backend

cd backend
npm install
npm run develop

### Frontendcd frontend
npm install
npm run dev
## Features
- User Registration & Login
- Skill Exchange System
- Rating & Reviews
- Email Notifications
- Admin Dashboard
- Content Pages (About, FAQ, Terms, Privacy)
